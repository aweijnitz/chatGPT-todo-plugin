# chatGPT-todo-plugin
Tutorial plugin for OpenAI/ChatGPT.

See https://platform.openai.com/docs/plugins/getting-started

Quickstart template Python (this project uses Node.js and Nest.js though): https://github.com/openai/plugins-quickstart

## Setup

	nvm use v18
	npm i -g @nestjs/cli # optional


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
